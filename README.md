# magicka.vim

![](magicka.jpg)

The `magicka` colorscheme for Vim

## Installation 
   * Stick the `magicka.vim` file in the folder `~/vimfiles/colors` (Windows) or `~/.vim/colors` (Linux/OSX)

## Configuration

Just edit your `~/_vimrc` (Windows) or `~/.vimrc` (Linux/OSX) to add:

    syntax on
    colorscheme magicka
